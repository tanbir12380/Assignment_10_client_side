Project name : BillHub

Live Link : https://rainbow-bunny-252a2b.netlify.app/

Description :
The BillHub is a MERN Stack-based web application that allows users to view and manage monthly utility bills such as Electricity, Gas, Water, Internet, and more. Users can securely log in, pay only the current month’s bills, and update their bill information as needed. The system provides a user-friendly interface and efficient backend to ensure smooth and reliable bill management.

Features :

- A full-stack utility bill payment system where users can view bills, pay bills, update payments, and manage their billing history.

- Frontend built with React, featuring responsive card layouts, custom modals, input validation, tooltips, and state-based conditional rendering.

- Backend developed using Node.js, Express, and MongoDB, providing bill CRUD operations, user data handling, and REST API endpoints.

- Authentication implemented with Firebase Auth, displaying the user’s profile photo and name within the interface.

- Frontend deployed on Netlify, with the backend deployed on Vercel.
